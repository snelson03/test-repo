# User Guide

## Overview

The Study Room Availability system is designed to be intuitive and self-guided.  
Instead of a traditional user manual, the application provides built-in guidance through tooltips, labels, and interactive UI elements.

---

## How to Use the Application

### Dashboard / Home

On the dashboard, you will find five separate containers:

- **Find a Room**  
  Navigates to the Find a Room page.

- **Campus Map**  
  Navigates to the Campus Map page.

- **Available Now**  
  Navigates to the Campus Map page.  
  Also displays the number of available rooms per building.

- **My Favorites**  
  Navigates to the Favorites page.  
  Displays your saved rooms and their current statuses.

- **Manage**  
  Contains:
  - Account
  - Groups
  - Notifications
  - Log Out  

---

### Finding a Room

- Navigate to **Find a Room**
- Select a building
- View available rooms using the color-coded system
- View the floorplan of the selected building

> **Note:**  
> Floorplan room colors are static and do not reflect real-time availability.  
> Refer to the room grid for live occupancy status.

---

### Understanding Room Status

- 🟢 Green → Available  
- 🔴 Red → Occupied  
- ⚫ Grey → Offline / unavailable  

---

### Using the Campus Map

- Open **Campus Map**
- Click or tap a building to view its rooms
- Double-click or use **Take Me There** to navigate directly

Additional interactions:
- Clicking a building in the legend highlights it on the map
- Clicking a pin shows the building name

---

### Adding Favorites

- Click the **heart icon** to favorite a room
- Click again to remove it

---

### Viewing / Removing Favorites

- Open the **Favorites** page
- Click a room to view details

To remove:
1. Click the **pencil icon**
2. Click the **trash icon** next to the room
3. Click the **checkmark icon** to confirm

---

### Room Details

Each room includes:

- Occupancy status  
- Location  
- Restrictions  

Restrictions indicate whether a room is:
- Public
- Group-restricted  

---

## Preferences

Use the dropdown to access:

- **Notifications**
  - Enable or disable in-app popup notifications

- **Account**
  - View email
  - Edit name
  - Edit phone number *(no functionality currently)*

- **Groups**
  - Join/manage groups *(no functionality currently)*

- **Appearance**
  - Light mode
  - Dark mode
  - System theme

---

## In-App Guidance

The application includes built-in help features:

- Tooltips explaining buttons and actions
- Visual indicators (icons, colors)
- Interactive UI feedback

These features reduce the need for a traditional manual.

---

## Notes

- Room availability updates automatically based on sensor input
- No manual refresh is required
